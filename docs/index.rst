Welcome to EXPLIoT's documentation!
===================================

Noun: /ɛkˈsplʌɪəti:/

**EXPLIoT** is a Framework for security testing and exploiting IoT products and
IoT infrastructure. It provides a set of plugins (test cases) which are used to
perform the assessment and can be extended easily with new ones. The name
**EXPLIoT** (pronounced *expl-aa-yo-tee*) is a pun on the word **exploit**
and explains the purpose of the framework i.e. IoT exploitation.

It is written in Python.
